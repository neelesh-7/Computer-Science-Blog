Strong & Emphasis:

  1. 1. <strong>
    The use of the <strong> element indicates that its content has strong importance.
    For example, the words contained in this element might be said with strong emphasis. By default, browsers will show the contents of a <strong> element in bold.

  2. 2. <em>
    The <em> element indicates emphasis that subtly changes the meaning of a sentence. By default browsers will show the contents of an <em> element in italic.



Quotations:
There are two elements commonly used for marking up

  1. 1. <blockquote>
    The <blockquote> element is used for longer quotes that take up an entire paragraph. Note how the <p> element is still used inside the <blockquote> Element.

  2. 2. <q>
    The <q> element is used for shorter quotes that sit within a paragraph.



There are three types of HTML lists:
Ordered, Unordered, and Definition.
  1. 1. Ordered lists use numbers.
  2. 2. Unordered lists use bullets.
  3. 3. Definition lists are used to define terminology.
  4. 4. Lists can be nested inside one another.



Lists:
There are two most common lists used in HTML and these are:
  1. 1. Ordered list.
    Ordered lists are lists where each item in the list is numbered. For example, the list might be a set of steps for a recipe that must be performed in order, or a legal contract where each point needs to be identified by a section Number.

  2. 2. Unordered list.
    Unordered lists are lists that begin with a bullet point (rather than characters that indicate order).



Ordered Lists:
  1. 1. <ol>
    The ordered list is created with the <ol> element.

  2. 2. <li>
    Each item in the list is placed between an opening <li> tag and a closing </li> tag. (The li stands for list item.)

Browsers indent lists by default.


Unordered Lists:
  1. • <ul>
    The unordered list is created with the <ul> element.

  2. • <li>
    Each item in the list is placed between an opening <li> tag and a closing </li> tag. (The li stands for list item.)



Nested Lists:
You can put a second list inside an <li> element to create a sub-list or nested list. Browsers display nested lists indented further than the parent list.
