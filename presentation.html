Our Website:

The Nutrition Planet website provides all the necessary information to attract our customers and give them an insight on our restaurant.


The Code:

We coded our website using Replit.com, a free software which lets you code in different languages. Our website is coded in HTML and CSS.


The Code in Detail:

  • ● Opening doctype and html tag <head> section; sets up the main page. Such as fonts display size and the look for the css file.

  • ● The head tags are important because they set the main looks



The Body:

  • ● This is the opening code of our website.

  • ● The header tags are nested in between the body tags

  • ● The navigation bar is coded using an unordered list wrapped with a href link



Home & About:

  • ● This code contains the home section.

  • ● The company name “Nutrition Planet” is wrapped with a span attribute; css coding was used to change colour.

  • ● After the “about us” description, there is a href attribute which links to the menu page.



The Menu:

  • ● The menu consists of three sections; Appetizers, Main Course, and Dessert.

  • ● Each item on the menu has an image to go with the details.

  • ● All the items on the menu are listed with the bx bx-cart class; all images can be edited through one css file.



Owners:

  • ● The owners section starts off with the heading “these are our founders!” followed by four individual cards for each owner.

  • ● Each card has a div with the class “card-background” followed by a style tag using the attribute “background-image” to indicate that the following image is the background image.

  • ● Each card has a span element with the class “card-content” which wraps the card heading (owner’s name) and card description (owner’s quote).

  • ● Each of the owner boxes are wrapped in an href attribute, which would refer the user to a section a bit more in depth about the owner.



Call to action:

  • ● Call to action uses a section with the class “cta” (call to action).

  • ● Within the section there is a heading with another saying of our company “We make quality food - Everyday” followed by an “Order Now!” button which would refer you to another area of the page where you would place your order.



Footer:

  • ● The footer section has an id of “contact” and contains all different aspects of our website the user can navigate and each form of contact we offer.

  • ● The footer section utilizes div tags with the class “col” to create 4 separate columns in CSS.

  • ● Each column uses an unordered list, the four lists being the menu links, contacts, information, and social medias.

  • ● Each of the list items use an href attribute which would be used to redirect the user to a separate page.

  • ● The menu links column uses a functional, simple navigation list similar to that of the navigation bar, but with less redesigning in CSS.
