Styling and Coloring!


“Clean code always looks like it was written by someone who cares.”
- Robert C. Martin


The HTML Style Attribute:

Setting the style of an HTML element, can be done with the style attribute.


Color Names:

In HTML, a color can be specified by using a color name.


Background Color:

You can set the background color for HTML elements. The CSS background-color property defines the background color for an HTML element.


Text Color:

You can set the color of text. The CSS color property defines the text color for an HTML element.


Border Color:

You can set the color of borders.


Color Values:

In HTML, colors can also be specified using RGB values, HEX values, HSL values, RGBA values, and HSLA values.


Fonts:

The CSS font-family property defines the font to be used for an HTML element.


Text Size:

The CSS font-size property defines the text size for an HTML element.


Text Alignment:

The CSS text-align property defines the horizontal text alignment for an HTML element.
