Announcement Program!


“First, solve the problem. Then, write the code.”
- John Johnson


3 screens (minimum):

  1. 1. Teacher Announcement Creator Page
    • ● Teacher can create the announcement and save it
    • ● Provide the facility for the teacher to enter 4 items of data:
      1. 1. GRADE
      2. 2. CATEGORY (e.g. gender)
      3. 3. CLUB
      4. 4. ANNOUNCEMENT DETAILS


  2. 2. Student Announcement Search Page
    • ● Student can search for announcements
    • ● It will allow the student to search using GRADE, CATEGORY and CLUB


  3. 3. Student Results Page
    • ● In this page, search results are displayed to the student
    • ● It will either display all the announcements that match with the search criteria or a friendly message to say there are no matching announcements
